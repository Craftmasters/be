INSTALLING
----------

See http://drupal.org/getting-started/install-contrib for instructions on
how to install or update Drupal modules.

Once Gmail Connector  module is installed and enabled,
you can adjust the settings for your
admin/config/gmail-connector

1.enable Gmail API access to get our API credentials.
2.visit Google’s Developer Console in our favourite web browser.
3.create a project (or choose an existing one) and go to the APIs section.
4.Choose “Gmail API” under the Google Apps APIs section and
click the “Enable API” button.

Now we need to create two sets of credentials,
one for an OAuth 2.0 client ID for a web application and
the other to create a browser API key.
This can be done in the credentials section of the Google Developer Console
by clicking the “Add Credentials” button.
